# MiGateWayMonitor
Mi GateWay Monitor(小米多功能网关监控系统) - https://blog.csdn.net/itas109

# NodeJS版本

Version : 1.0.4.191201

## 功能
- [x] 配置文件动态修改
- [x] 定时器任务（心跳包）
- [x] 邮件、钉钉、日志等消息记录
- [x] 看门狗异常重启
- [x] 记录连续运行时间
- [ ] 日志文件自动删除
- [ ] 微信推送
- [ ] 短信推送
- [ ] 电话推送

## 运行

```
cd src\MiGateWayMonitorJS

npm install

npm start
```
